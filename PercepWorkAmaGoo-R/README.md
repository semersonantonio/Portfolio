## Employee Reviews Text Analysis  
### Data Science & NLP Project (R)

#### Overview
This project demonstrates how to apply NLP techniques to real-world unstructured data.  
Employee reviews from Amazon and Google were analyzed to extract insights related to organizational culture, work environment, and employee sentiment.

#### Data Science Responsibilities
- Data ingestion and cleaning
- Text preprocessing and normalization
- Feature extraction using TDMs
- N-gram modeling (uni/bi/trigram)
- Exploratory analysis and visualization
- Insight generation and storytelling

#### Key Insights
- Amazon’s culture is perceived as high-performance and fast-paced.
- Google is associated with better work-life balance and employee perks.
- Cultural factors strongly influence employee satisfaction.

#### Stack
R, NLP, Text Mining, Data Visualization
